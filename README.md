# StockTradingData

This repo provides a script and a list of tickers that I'm currently invested in. 

To recreate on your machine, do the following:

1. Create an instance of postgres, configuring a schema matching the naming convention I've provided in this script.
2. Create a user (you can use postgres, it's just local anyways... right?)
3. Update your variables in the .env to point to your database.
4. Update your tickers to stocks you may care about.
5. Download powerbi and open the provided file.
6. Update your tickers to point to your connected data source.

Bam! You now have 60 days of up-to-date stock information and some handy charts to show you when is best to buy. (The bottom, NOT the top)
